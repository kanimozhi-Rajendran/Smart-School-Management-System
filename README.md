# 🎓 Smart School Management System

A full-stack School ERP web application developed using **React, Node.js, Express.js, and MySQL**. The system provides separate portals for **Admin**, **Teacher**, and **Student** with secure role-based authentication and modern UI.

---

- Fees Management

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/Kanimozhi-Rajendran/Smart-School-Management-System.git
```

### Backend

```bash
cd backend
npm install
node scripts/initDb.js
node index.js
```

Backend runs on:

```
http://localhost:5000
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
